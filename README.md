🚀 MyTurboVPN v1.0
Ushbu dastur Windows operatsion tizimi uchun mo'ljallangan, yuqori tezlikdagi HTTP/HTTPS proksilarni topuvchi va boshqaruvchi monitoring vositasidir. Dastur minglab ochiq proksilar ichidan eng tezkor 20 tasini saralab beradi va tizimga xavfsiz integratsiya qiladi.

✨ Asosiy Imkoniyatlar
Avtomatik Skannerlash: Har 30 soniyada proksi bazalarini yangilaydi.

Tezlik bo'yicha Saralash: Faqatgina eng past ping (kechikish)ga ega bo'lgan "chaqqon" proksilarni ko'rsatadi.

Davlatni Aniqlash: Har bir IP manzilning geografik joylashuvini (mamlakatini) aniqlab beradi.

Faqat HTTP/HTTPS: SOCKS protokollaridan voz kechilib, faqat barqaror HTTP ulanishlar bilan ishlaydi.

Xavfsiz Chiqish (Auto-Cleanup): Dastur yopilganda (X tugmasi) barcha proksi sozlamalari avtomatik tozalanadi va tizim o'z internetingizga qaytadi.

Alohida Boshqaruv: Bir tugma bilan proksini yoqish yoki butunlay o'chirish imkoniyati.

🛠 Texnologiyalar
Til: Python 3.x

Interfeys: PyQt6

Parallelizm: ThreadPoolExecutor (100+ oqimda parallel tekshirish)

Tizim: Windows Registry (winreg) orqali internet sozlamalarini boshqarish

🚀 Ishga tushirish
Kerakli kutubxonalarni o'rnating:

Bash
pip install PyQt6 requests
Dasturni ishga tushiring:

Bash
python main.py
⚠️ Muhim Eslatma
Dastur yopilayotgan vaqtda Windows internet sozlamalarini asl holiga qaytaradi. Bu sizning internetingiz "proksi yoqilgan holatda qolib ketishi" natijasida yuzaga keladigan nosozliklarning oldini oladi.
